# My Project

A static HTML/CSS/JS project.

## How to run

The **Start application** workflow serves the files with Python's built-in HTTP server:

```
python3 -m http.server 5000
```

Open the **Preview** pane to see `index.html`. Any changes you save to `index.html`, `style.css`, or `script.js` will appear after a browser refresh.

## Files

- `index.html` — page structure
- `style.css` — styles
- `script.js` — JavaScript
